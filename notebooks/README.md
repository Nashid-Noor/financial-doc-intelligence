# Financial Document Intelligence - Notebooks

This directory contains Jupyter notebooks for exploration and evaluation.

## Notebooks

1. **01_data_exploration.ipynb** - Explore financial Q&A datasets
2. **02_chunking_experiments.ipynb** - Experiment with different chunking strategies
3. **03_retrieval_evaluation.ipynb** - Evaluate retrieval performance
4. **04_model_evaluation.ipynb** - Evaluate fine-tuned model

## Running Notebooks

```bash
jupyter notebook
# or
jupyter lab
```

To create the notebooks, run:
```python
jupyter notebook
```

Then create new notebooks following the naming convention above.
